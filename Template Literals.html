JavaScript - Template Strings

Things to Know

  • According to MDN: Template literals are literals delimited with backtick (`) characters, allowing for multi-line strings, string interpolation with embedded expressions, and special constructs called tagged templates.
  • Template literals are used to embed expressions inside a string.
  • These are sometimes called as template strings
  • A back tick is used for the purpose
  • To avoid the backtick or the dollar sign, use a back slash

Syntax

`string text` `string text line 1 string text line 2` `string text ${expression} string text` tagFunction`string text ${expression} string text`
